# Handoff Spec: Pip v2 — "The Bar Is Home"

Generated by /design-handoff on 2026-07-11 for the Shrubby landing site.
Reference: `Downloads/animate_this_ladybug_for_a_web.mp4` (10s, 1280×720 — sculptural
3/4-view clay ladybug on studio floor). Supersedes the side-view character sheet in
the Pip v1 plan; the behavior brain, transform channels, gating, and debug API from
`src/ladybug.tsx` carry over unchanged unless noted.

### Overview
Pip stops commuting between the bar and the meadow: **the footer glass pill is home.**
The character is re-sculpted to the reference's 3/4 clay body, and the act repertoire
is rebuilt around physical comedy WITH the bar itself — its rim, its rounded ends, its
links, its letters. Engagement goal: the visitor's cursor is part of Pip's world; the
bug notices, reacts, and baits interaction. Meadow flights still exist but as rare
punctuation (≤15% of acts), not the main show.

### Why (the one paragraph the implementer must not skip)
v1 read as a critter that *visits* the footer. v2 is a resident with a territory. All
comedy derives from one premise: **Pip treats the bar as furniture** — a tightrope, a
slide, a mirror, a bed. Every act must involve the bar's real geometry (measured rects,
never constants), because the joke only lands if physics feels true.

### Character v2 (re-sculpt to reference)

| Dome material | Matte clay radial: `#C86A47 → #B9502F → #7E2E17` (existing family), highlight ellipse ≤ .14 opacity, top-left |
| Dots | 5 visible, soft-edged (2% blur via SVG feGaussianBlur? NO — CSP-safe: use a radial gradient per dot `#22371D → #1A2814`), sizes 8–14% of dome |
| Head | Near-black `#161F12` gloss: ONE crisp white catchlight (2px, fixed). Eyes: two small (60% smaller than v1) — expressiveness moves to ANTENNAE + body language |
| Antennae | Ball-tipped, 2.4px stroke, the PRIMARY emotional instrument: perk (alert), droop (sleepy), swivel toward cursor (noticing), snap back (startled) — each a named CSS class, 160–300ms ease-out |
| Shadow | Contact ellipse, `rgba(14,32,21,.38)` radial fade — reference shows tight contact shadow; keep bottom -7px, width 72% |
| Size | 64px desktop / 48px mobile (slightly larger than v1 — the dome silhouette needs it) |
| ViewBox | 0 0 80 64; feet baseline at y=60 |

### The v2 act repertoire (bar-centric; weights sum ≈ 100)
| # | Act | w | Choreography (3 beats: anticipation / action / settle) |
|---|---|---|---|
| 1 | tightrope | 14 | walk top rim with ±6° wobble oscillation (240ms cycle); 20% chance mid-way: one foot slips — catch with wing-flare (120ms crouch / 340ms SETTLE) |
| 2 | rim-sit + people-watch | 12 | settle at rim edge, antennae track cursor x (lerp 8%/frame, ±30°); blink pattern doubles |
| 3 | end-cap slide | 8 | climb to cap top (400ms), pause 600ms (anticipation!), slide down the pill's 100px-radius curve 700ms ease-in, land squash scaleY .78, look around "did anyone see that" (= v1 lookAround) |
| 4 | upside-down hang | 7 | walk off bottom rim → clip-path reveal below bar, hang by 2 legs, sway ±4° 1.2s cycle, 3–6s, pull-up return (600ms) |
| 5 | peek-a-boo | 8 | duck behind bar (z:1, glass blurs body), head pops above rim at RANDOM x (3 pops, 500ms apart, different spots — the unexpected bit) |
| 6 | letter-sit | 6 | sit on the "y" descender of "Shrubby" (brand span rect + 0.82·width offset), antennae droop content |
| 7 | glass-squeegee | 5 | face viewer, one foreleg wipes an arc on the glass (3 strokes, 420ms each, rotate −15°→25°), inspect, nod |
| 8 | arrow-nudge | 5 | walk to "Read the making →", push the "→" with head (2 shoves, 280ms, whole bug recoils 3px), look at camera (CTA beat successor — keep once/session cap) |
| 9 | measure-the-bar | 4 | pace rim end-to-end in exaggerated tip-toe steps (scurry at 0.4× speed, high-step 40% leg lift), stop, antennae "calculate" swirl 800ms |
| 10 | micro-nap | 6 | mid-walk slow-down over 1.2s → lids close → 2.5s sleep with existing zzz → slide 6px along rim → startle-wake (tap-wake spec) |
| 11 | cursor-shy approach | 8 | if pointer idle ≥2s within 160px: creep toward it 30% distance, stop, retreat 10%, repeat — never reaches (bait loop) |
| 12 | startle scatter | 6 | pointer enters <56px while Pip mid-act: interrupt (commitStyles capture per v1), 90ms antennae snap + hop away 40–70px along rim |
| 13 | meadow flight | 9 | v1.1 spline flight (monotonic x, 3D pose) — now RARE; always returns to a bar anchor |
| 14 | flip-fail | 2 | attempts to fly from under-rim hang, wings tangle, drops 12px, catches rim — one per session max |

Scheduler unchanged (3–14s × tempo 0.65–1.6; single timeout; almanac re-eval per tick;
weights zeroed for acts whose geometry is unavailable — e.g. no-cap-slide under 640px wrap).

### States and Interactions
| Element | State | Behavior |
|---|---|---|
| Pip button | tap ×1 / ×2 / ×3 (5s window) | lookAround / startle-hop / grumpy fly-off returning ≤10s (v1 spec, unchanged) |
| Pip button | tap while hanging | falls, lands on feet, indignant ruffle (NEW — overrides escalation once) |
| Pip button | keyboard focus | leaf ring on pose (existing); Enter/Space = tap |
| Pip | REDUCE motion | static rim-sit + correct almanac prop; tap = opacity pulse |
| Cursor | idle in footer ≥2s | act 11 eligible; antennae tracking (act 2) always-on when resting |
| Links | any time | Pip never RESTS on link rects; flight/walk crossings ≤ 500ms occlusion |
| Bar | width < 50px (harness/collapse) | static perch fallback (v1 guard) |
| Almanac | all v1 wardrobe/holiday/season/weather + sleep rules apply on the bar | poppy = calm day (no comedy acts 1,3,5,7,8,14) |

### Responsive Behavior
| Breakpoint | Changes |
|---|---|
| >640px | full repertoire; 64px bug |
| ≤640px (bar wraps, radius 26px) | 48px bug; disable end-cap slide (no round caps) + letter-sit (wordmark too small); rim = top edge of wrapped block; cursor acts replaced by touch-only escalation |
| Any resize | re-measure anchors at next landing (never cached — v1 rule) |

### Edge Cases
- **Interrupt mid-act**: rejecting act-waits + commitStyles capture + finally-cleanup (v1.2 mechanics — reuse `flushWaits`/`interruptAll`; every new act with props/classes gets try/finally).
- **Long/i18n bar text**: all geometry from `field__brand` / link rects — no hardcoded x.
- **Two bugs never**: swarm minis (5-tap egg) are pointer-events:none and self-remove (v1).
- **Empty/error**: bug mounts perched; measurement failure = static perch; never blank.
- **Timers**: zero when footer off-viewport or tab hidden (IO + visibilitychange, v1).

### Accessibility Notes
- One focusable element: the Pip `<button aria-label="Pip the ladybug — give a tap">`. Acts never move focus.
- Decorative layers (shadow, props, minis, petals) `aria-hidden`; no ARIA live regions (silent comedy).
- Hit target ≥44px incl. while hanging (extend `::after` inset −8px downward when under-rim).
- `prefers-reduced-motion`: full teardown to static rim-sit (v1 listener), wardrobe intact.
- Focus order unchanged: bar links before Pip (Pip is appended after links in DOM).

### Engineering contract (unchanged from v1 — implementer MUST keep)
Transform channels (.pip translate / .pip__pose facing+bank+squash / .pip__body breathe /
limbs CSS), `composite:'add'` forbidden, commitStyles-before-cancel, onfinish not awaited
.finished, Catmull-Rom monotonic-x flights, measured-rect anchors + `.sprout-top` exclusion,
node:test almanac suite must stay green, `window.__pip` debug API drives all QA (hidden
preview panes freeze WAAPI timelines — scrub `currentTime`; setDate parses Y/M/D locally).

### Implementation order
1. Re-sculpt SVG (viewBox 80×64, 3/4 anatomy, antennae rig as separate groups) + size bump.
2. Bar-geography module: rim path, caps, under-rim clip, letter/link/no-rest rects.
3. Antennae expressiveness system (classes + cursor tracking rAF, gated like particles).
4. Acts 1–10 (bar physical comedy), reusing v1 primitives (play/wait/walkTo).
5. Cursor sensing (acts 11–12) — pointermove on `.field`, throttled, gated.
6. Re-weight scheduler; flights to w=9; wire flip-fail session cap.
7. QA per v1 matrix + new: link-occlusion timing, hang hit-target, mobile disables.
